# API Testing Framework (Postman)

## 📌 Overview
This project demonstrates an end-to-end API Testing Framework built using Postman. It covers authentication, CRUD operations, validation techniques, and data-driven testing for a CRM-like application.

---

## 🚀 Features

- 🔐 Authentication (Login + Token Handling)
- 🔁 CRUD Operations (Create, Read, Update, Delete)
- ⚙️ Environment-based configuration (Dev & Prod)
- 🔄 Dynamic data generation using pre-request scripts
- 📊 Data-driven testing using CSV
- 🧪 Automated test scripts (status, response, headers)
- 📐 JSON Schema validation
- ⏱️ Response time validation
